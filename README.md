# Godot 3D Prototype

A 3D prototype game created using Godot Engine.

## Features
- Character animation
- Scene management
- GDScript logic
- Asset imports (GLB models, textures)

## How to Run
1. Install Godot Engine
2. Clone this project
3. Open `project.godot`

## Project Structure
- `/Charecter` → 3D models
- `/Env` → environment textures
- `/sound` → SFX
- `/script` → GDScript
